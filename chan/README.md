## 总结
读已关闭的chan返回类型0值,如果有第二个返回值,返回是否关闭bool值

写已关闭的chan抛出异常,可以defer recover处理,不推荐这种实现,可以通过只有单协程关闭解决问题,其他写协程通过ctx(ctx.Done())+select停止写入
